## How to run project
Step 1: Clone my project.\
Step 2: Press npm i to install node packages in terminal.\
Step 3: Run project by command npm start. Enjoy!\
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.\

### Features

Add robo.\
Delete robo.\
Edit & Update robo.\
View list robo.\
